Chrome Logger

Posted on Aug 07, 2013 by Leonardo Zhou · Leave a comment

安装好Chrome Logger的Chrome扩展，并在服务器端安装对应语言（目前支持Ruby, PHP, Python, Node.js, .NET）的Chrome Logger库后，就可以在服务器端用类似console.(variable)的语句log变量，然后在Chrome Dev Tools上查看变量的实际值。

以Django为例, 配置十分简单：

# django middleware
MIDDLEWARE_CLASSES = (
    'chromelogger.DjangoMiddleware'
)

# views
import chromelogger as console
console.log('Hello console!')
console.get_header()
